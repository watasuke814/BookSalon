## UI,UXの工夫
- 直感的に使いやすいようにリンクを少なめに配置
- デザインの統一感を考え、背景は暗めの本棚で合わせる
- 主に黒,白,背景の三色のみ
- トップページにおいてスクロールでページが変わる機能

## 言語,プラグイン等

- Ruby on Rails 5.1.6.1
- html5
- css3
